Aid Badges is a lightweight, distributed system for coordinating humanitarian aid efforts. Community-based organizations are able to create requests for volunteer support throughout the Aid Badges API (via a web interface or SMS). Volunteers can search for aid opportunities—labor or supplies—nearby.

OpenBadges are awarded to recognize volunteer contributions and skills.
